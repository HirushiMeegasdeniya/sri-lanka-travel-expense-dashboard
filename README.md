# Sri Lanka Travel Expense Intelligence Dashboard

This is a beginner-friendly data science project that analyzes travel expenses across popular Sri Lankan destinations.

## Project Goals

- Generate a realistic travel expense dataset
- Clean and prepare the data
- Analyze travel spending patterns
- Create visualizations
- Build an interactive Streamlit dashboard
- Publish the project on GitHub and LinkedIn

## Tools Used

- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Plotly
- Streamlit
- GitHub
